# JHelper

JHelper는 전북대학교 학우들을 위한 Computer Science 학습 플랫폼인 JEdutools의 도움말 페이지의 배포를 위한 시스템입니다.

JHelper는 Github의 Public Project로 운영되며 누구나 Contribution 할 수 있습니다.

[레포지터리 링크](https://github.com/JBNU-JEduTools/document) - 클릭 시 이동

# Contribution

Contribution은 [개발자 매뉴얼](https://jbnu-jedutools.github.io/document/JHelper/devManual)을 참고바랍니다.

# 버그 제보 및 기능 제안

기능 제안은 Repository 내의 Issue 기능을 통해 제안해 주세요 - [바로가기](https://github.com/JBNU-JEduTools/document/issues)

# 원본 프로젝트

JEdutools 문서화 프로젝트는 codebushi의 gatsby-theme-document-example를 수정해 사용하고 있습니다.
```
git clone https://github.com/codebushi/gatsby-theme-document-example
```
