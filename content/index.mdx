### JEduTools

JEduTools는 전북대학교의 SW 교육을 위한 도구 모음입니다.

| 이름   | 소개   | 링크   | 도움말 | GitHub 저장소
|---------|---------|:---------:|:---------:|:---------:|
| JCloud  | 전북대학교 클라우드 시스템  | [Click](http://jcloud.jbnu.ac.kr)   |[Manual](https://jcloud-devops.github.io/) | [repo](https://github.com/JBNU-JEduTools/JCloud)   |
| Litmus   | 프로그래밍 과제 자동 채점 시스템   | [Click](http://litmus.jbnu.ac.kr)   | |  |
| JCode   | Web 기반 IDE 서비스   |  | [Manual](https://jbnu-jedutools.github.io/JHelper/JCode) | [repo](https://github.com/JBNU-JEduTools/JCode) |
| JTMI   | SW 관련 QnA 게시판   | [Click](http://idea.jbnu.ac.kr)   | [Manual](https://jbnu-jedutools.github.io/JHelper/JStackOverflow) | [repo](https://github.com/JBNU-JEduTools/JStackoverflow)   |
| JHelper   | JEduTools 도움말 작성 및 배포 시스템   | [Click](https://jbnu-jedutools.github.io/JHelper/)   | [Manual](https://jbnu-jedutools.github.io/JHelper/JHelper) | [repo](https://github.com/JBNU-JEduTools/JHelper)  |
| Watcher | JCode 기반 코딩 액티비티 수집 및 분석 시스템   |    |   | [repo](https://github.com/JBNU-JEduTools/Watcher)|

* JEduTools는 SW중심대학사업단의 지원으로 컴퓨터인공지능학부에서 운영하고 있습니다.
* JCloud는 정보혁신처 (가칭)클라우드관리팀에서 운영하고 있습니다.
